Referência rápida Jinja2

{% raw %} 
  {{ variavel }}               → exibe valor (escapado)
  {{ variavel | filtro }}      → aplica filtro (ex: | upper, | length)
  {% if condicao %}            → condicional
  {% elif outra %}             → senão se
  {% else %}                   → senão
  {% endif %}                  → fecha if

  {% for item in lista %}      → loop
  {% endfor %}                 → fecha for
  loop.index / loop.index0     → posição no loop (1-based / 0-based)

  {% block nome %}             → define área sobrescritível
  {% endblock %}               → fecha bloco
  {% extends "base.html" %}    → herda layout de outro template
  {% include "arquivo.html" %} → inclui fragmento reutilizável

  {# comentário #}             → comentário Jinja2 (não aparece no HTML)
{% endraw %}
